capacity auto-approve

PRISHTINË · 2026
feat/capacity-auto-approve
Version 1.0

Le toggle capacity_auto_approve permet aux salons en mode capacité de bypasser l'étape de validation manuelle. Quand il est actif, chaque nouveau rendez-vous passe directement au statut Confirmé — sans action de l'owner.

Deux points de contact UI : l'écran "À confirmer" qui reflète l'état actif avec un message éditorial, et le toggle dans les Paramètres de Mon Salon, conditionnel au mode de réservation.

  • 1
    Menu toujours visible L'entrée "À confirmer" reste dans la navigation même quand auto-approve est ON. On change le contenu, pas la structure — cohérence du dashboard owner.
  • 2
    Toggle conditionnel Le paramètre est visible uniquement pour les salons en mode capacity_based. En mode employee, il est caché — pas de confusion.
  • 3
    Ton éditorial, non alarmant L'empty state informe sans alarmer. Pas d'icône générique Material — une marque calligraphique fine, un titre Fraunces italique, un bouton outline sobre vers le paramètre.
